Playback

After you have created and named the clip, use the name as a command to tell Voice Blaster Platinum to play it back. For example, if you named your clip rad, use the command ;rad to hear it.

Other fun ones

  • ;clip (user) - select a segment to save from the last 30 seconds of audio
  • ;delete (title) - delete a clip permanently. for example, after ;delete rad, the command ;rad will do nothing. only an admin can do this
  • ;gtfo - force the bot to leave the voice channel it's in. Only an admin can use this
  • ;list - list all the clips you can play
  • ;listen - this will undo ;stop listening. Only an admin can use this
  • ;nuke - clear all queued-up clips. Useful after someone sends ;repeat rad 9001 times. Only an admin can use ;nuke
  • ;random - play a random clip
  • ;repeat (title) (number) times - hear a clip multiple times. For example, ;repeat rad 69 times if you really liked hearing ;rad
  • ;stop listening - stops the bot from listening to your chat until you send ;listen. Only an admin can use this
  • ;welcome (user) with (clip) - play the clip whenever the user joins voice chat

Housekeeping

  • ;admin (user) - give someone admin privileges. Only the guild owner can do this
  • ;ban (user) - prevent a user from interacting with Voice Blaster Platinum. Only an admin can do this
  • ;share - get the bot add link
  • ;sync - tell Voice Blaster Platinum to download new clips, in case it missed one. You shouldn't have to do this, but nobody's perfect
  • ;unadmin (user) - undoes ;admin
  • ;unban (user) - undoes ;ban
